### 下载与使用方法
在[发行界面](https://github.com/Yamikani-Flipped/LingYun-Class-Widgets/releases)，下载最新版本，按照指引进行安装，安装完成后运行桌面上的 凌云班级组件 2 即可使用。

可在系统托盘中找到凌云图标，进行设置、息屏等操作。

[点击这里](https://www.yamikani.cn/docs/) 查看官方使用文档。

## 远程喊人、远程操作、远程喊人： [service.yamikani.cn](service.yamikani.cn)

### 本软件主要功能和特性
- 桌面显示时间、日期
- 支持展示课表、值日表、倒计时和当前课程
- 考勤管理
- 远程喊人、远程操作、远程喊人,集中管理班级
- 适合在自习、专注情况下使用的“息屏显示”组件
- 插件系统
- 支持集中控制，大部分设置项均支持云同步，完善的多班管理
- 支持客户端内快捷更新
- 便捷的课表、时间线编辑
- 支持[**CSES通用课程表格式**](https://github.com/SmartTeachCN/CSES)（Course Schedule Exchange Schema）
- 遇到节假日等等情况的调课管理
- 上下课提醒、预备铃提醒
- 支持存储多个课表文件，并且可以快速导入导出和切换
- 软件高度定制化，大部分功能支持设置中修改
- ... （待完善）
